# Lang Tutor

Персональный AI-тьютор для изучения иностранных языков. Чат с языковой моделью, которая ведёт уроки в выбранном формате, вводит новые слова и отслеживает прогресс через интервальное повторение (SM-2).

Поддерживаемые форматы урока: повторение, новая лексика, грамматика, чтение, письмо, разговорная практика.

---

## Получение API-ключа

Приложение использует языковые модели через [OpenRouter](https://openrouter.ai) — агрегатор, который даёт доступ к сотням моделей через одно API.

1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai).
2. Перейдите в раздел **Keys** → **Create Key** и скопируйте ключ (он начинается с `sk-or-...`).
3. Бесплатный лимит позволяет начать без привязки карты; для интенсивного использования пополните баланс.

Ключ нужно вставить в файл `.env.local` (см. шаг 2 ниже).

## Запуск в режиме разработки

> Требуется **Node.js 18**. На более новых версиях `better-sqlite3` может не собраться.

```bash
# 1. Установить зависимости
yarn install

# 2. Создать .env.local и добавить ключ
cp .env.example .env.local
# Открыть .env.local и заменить 'your_key_here' на свой ключ:
# OPENROUTER_API_KEY=sk-or-...

# 3. Создать директорию для базы данных
yarn setup

# 4. Запустить
yarn dev
```

Приложение будет доступно на http://localhost:3000.

---

## Запуск через Docker

Требуется Docker и Docker Compose.

```bash
# Убедиться, что .env.local с OPENROUTER_API_KEY существует (см. раздел «Получение API-ключа»)
# Затем:
docker compose up --build
```

Приложение поднимется на http://localhost:3000. База данных хранится в папке `./data` на хосте и переживает перезапуск контейнера.

Остановить:
```bash
docker compose down
```

## Как пользоваться приложением

### 1. Добавить язык
При первом запуске автоматически откроется мастер настройки. Выберите язык, укажите уровень (или пройдите автоматический тест) и выберите интересующие темы. После этого можно добавлять другие языки через кнопку **+ Добавить язык** в сайдбаре.

### 2. Выбрать формат урока
В сайдбаре выберите один из форматов:
- **Повторение** — повторение слов из очереди SM-2
- **Новая лексика** — тьютор вводит новые слова и проверяет их использование
- **Грамматика** — объяснение правила и упражнения
- **Чтение** — короткий текст и вопросы на понимание
- **Письмо** — задание на составление текста с разбором ошибок
- **Разговорная практика** — ситуативный диалог

Тьютор сам начинает урок и ведёт его до конца. После каждого ответа он оценивает его и сразу даёт следующее задание.

### 3. Следить за прогрессом
В нижней части сайдбара отображается статистика: изученные слова, слова для повторения сегодня, дни подряд и количество сессий. Нажатие на карточку **слов** или **повторить** показывает соответствующий список прямо в чате.

Слова с низкими оценками (< 5/10) отображаются в разделе **Слабые места**.
